# 数据库说明文档

## 1. 项目概述

本项目为 **智能仓储与物流协同系统 (Smart Warehouse & Logistics Collaboration System)** 的后端数据库设计。

数据库设计严格遵循 **事件驱动架构** 与 **微服务模块化** 的理念，支持从入库、在库管理、出库拣选到物流追踪的全链路数字化流程。

- **数据库名**: `warehouse`
- **字符集**: `utf8mb4`

------

## 2. 模块划分与命名

为了模拟微服务架构的数据隔离，表名采用了前缀区分：

| **模块前缀**     | **模块名称** | **功能描述**                         | **核心表举例**                              |
| ---------------- | ------------ | ------------------------------------ | ------------------------------------------- |
| **sys_ / base_** | 基础服务     | 用户权限、货物主数据、仓库基础设施工 | `sys_user`, `base_goods`, `base_robot`      |
| **wms_**         | 仓储服务     | 核心库存管理、入库单、出库单、盘点   | `wms_inventory`, `wms_inbound_note`         |
| **oms_**         | 订单服务     | 客户销售订单管理                     | `oms_sales_order`                           |
| **tms_**         | 物流服务     | 运输单管理、物流轨迹追踪             | `tms_transport_order`, `tms_transport_node` |
| **iot_**         | 物联网服务   | 传感器数据采集与环境监控             | `iot_sensor_data`                           |
